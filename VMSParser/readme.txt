VMSParser

Prerequisites:
- Visual Studio 2010 Express or Pro
  http://www.microsoft.com/visualstudio/en-us/products/2010-editions/express
- Boost 1.48.0
  http://www.boost.org/users/history/version_1_48_0.html

This project contains a VMS sentence parser that converts the messages to CSV, TSV, or KML file formats.  To start on this project, simply run the build_project.bat file in the build directory, open up the VS2010 project solution, and build to build the executable.
